# BNC Builders - Digital Marketing Proposal

Complete digital marketing infrastructure proposal for BNC Builders Inc., powered by Astro and deployed on Vercel.

## Project Overview

This documentation site contains the complete proposal for BNC Builders including:

1. **Executive Summary** - Overview of the complete marketing takeover
2. **Current State Analysis** - Where BNC Builders is today
3. **Website Migration** - Moving off Scorpion to a modern, owned platform
4. **Tracking & Analytics** - Enterprise-grade measurement setup
5. **Paid Advertising** - Google Ads + Meta Ads strategy
6. **SEO Strategy** - Dominate San Diego County search results
7. **Custom CRM Build** - Purpose-built lead management system
8. **Timeline & Investment** - Complete project timeline and pricing
9. **CSR Software Implementation** - Unified communication and workflow system

## Scope

### What We're Building

| Component | Description |
|-----------|-------------|
| Website | Modern, fast, SEO-optimized site you own |
| CRM | Custom lead management with offline conversion tracking |
| CSR Software | Unified inbox, automated follow-ups, production tracking |
| Tracking | GA4, GTM, CallRail, offline conversions |
| Paid Ads | Google Ads + Meta Ads management |
| SEO | Technical SEO, content, local optimization |

### CSR Pain Points Addressed

- All communication in one place (text, email, calls)
- Full communication history per lead
- Automated reminders & workflows
- CSR production tracking
- Templates for all messages
- Texting from office number
- Automatic lead import from all platforms
- Unified follow-up system
